# Description GitHub (pour les settings du repo)

🎪 Web app mobile-first pour organiser un week-end festival entre potes en Ardèche. Programme, chat, galerie, dépenses partagées, jeux mystères, tournoi billard.

## Option longue (pour "About" ou description détaillée)
App web Next.js + Supabase pour le "Cros-Chella" — un week-end festif entre amis (31 juil - 2 août 2026, Ardèche). Features : profils custom, programme collaboratif, chat temps réel, galerie photos, tricount/dépenses, jeux mystères, tournoi billard, badges, notifications realtime. Dark theme festival, mobile-first, glass morphism.
