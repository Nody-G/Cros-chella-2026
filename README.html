✨ Fonctionnalités

Tout ce qu'il faut pour un week-end mémorable — et même ce qu'on n'a pas demandé.

👤

Participants & Profils

Profils personnalisés avec emoji avatar, pseudo, bio, rôle de festival, spécialité, super-pouvoir, faiblesse et phrase fétiche. Badges de compatibilité alcool/fume entre potes.

📋

Programme

Planning du week-end (jeudi → dimanche) avec timeline visuelle. CRUD admin, propositions par les participants, votes, commentaires et photos sur chaque activité.

🎮

Jeux Mystères

Soumission secrète de jeux par chacun (Quiz, Défi physique, Défi alcool, Dégoûtant, Culture, Créatif). Révélés par l'admin au bon moment. Teasers avant la révélation.

💬

Chat Temps Réel

Mur de discussion avec réactions emoji, modification/suppression (appui long mobile), images avec overlay edit/delete, vibration sur nouveaux messages, horodatage intelligent.

📸

Galerie Photos

Upload avec compression, commentaires, viewer plein écran avec swipe tactile gauche/droite, likes, CRUD complet, lien Google Drive pour photos/vidéos HD.

💰

Dépenses Partagées

Tricount intégré — suivi des frais partagés, catégories, répartition automatique, calcul des balances et règlements optimaux entre participants.

🎱

Tournoi de Billard

Tournoi par équipes avec bracket automatique, suivi des scores et classement en temps réel.

🏅

Badges Custom

Badges décernés par l'admin (emoji + titre + description). Visibles sous chaque participant, cliquables sur mobile.

📊

Sondages

Votes en temps réel avec barres de résultats animées et compteur pulse. Décisions collectives instantanées.

🍺

Alcool & Inventaire

Page dédiée avec préférences de chacun et inventaire absurde "Botardèche". Parce que les priorités, c'est important.

🔔

Notifications Push

Pastilles rouges en temps réel + Notifications Push natives (PWA) en tâche de fond. Ne ratez aucun message du chat.

🐛

Feedback

"Signaler un bug ou proposer une idée" — feedback anonyme avec statuts persistants pour l'admin. L'amélioration continue, version festival.

📱 Screenshots

Un aperçu de l'app en conditions réelles — dark theme festival, glassmorphism et ambiance nuit.

🚀 Installation

Lancer le projet en local.

# Cloner le repo
git clone https://github.com/Nody-G/Cros-chella-2026.git
cd Cros-chella-2026

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.example .env.local
# Remplir SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_DB_URL

# Lancer en dev
npm run dev
